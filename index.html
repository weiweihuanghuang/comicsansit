Comic Sans It

Just imagine that your favorite website
is written in Comic Sans

Sounds like cruel horror for you? Well, congratulations, you're one of thousands who don't like the Comic Sans Font and its further implementing into ad-campaigns and stuff. As ComicSansCriminal perfectly describes, »Comic Sans was made for the little comic book style help speech bubbles in programs like Microsoft Word«. That's fact. But it has become very popular over the last years, so that a lot of serious institutions like hospitals, churches, restaurants or even police stations use this comical font nowadays.

This bookmarklet will show you how your favorite website will look like when it's written in Comic Sans - completely.
The following javascript function accomplishes the css injection when initialized:

javascript: (function () {
    var fileref = document.createElement('link');
    fileref.setAttribute('rel', 'stylesheet');
    fileref.setAttribute('type', 'text/css');
    fileref.setAttribute('href', 'http://kaibrueckers.github.io/comicsansit/comicsansit.css');
    document.getElementsByTagName('head')[0].appendChild(fileref);
})();

Just drag this bookmarklet below and drop it to your browser bookmarks toolbar. Every time you click it now, the current website in your browser will change (not working on most SSL-encrypted websites, though).
